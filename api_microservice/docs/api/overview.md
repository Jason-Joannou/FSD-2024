# API Overview

Welcome to the API documentation for the FSD API. This document provides an overview of the available endpoints and their usage.

## Endpoints

- [GET /](#get--): Returns a welcome message.
- [POST /line_graph](#post--line_graph): Generates a line graph.
- [POST /bar_graph](#post--bar_graph): Generates a bar graph.
- [POST /scatter_plot](#post--scatter_plot): Generates a scatter plot.
- [POST /histogram](#post--histogram): Generates a histogram.
- [POST /pie_chart](#post--pie_chart): Generates a pie chart.
- [POST /box_plot](#post--box_plot): Generates a box plot.
- [POST /heatmap](#post--heatmap): Generates a heatmap.
- [POST /area_chart](#post--area_chart): Generates an area chart.
- [POST /bubble_chart](#post--bubble_chart): Generates a bubble chart.
- [POST /candlestick_chart](#post--candlestick_chart): Generates a candlestick chart.
